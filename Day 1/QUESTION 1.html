CONSOLE

Explore and explain the various methods in console function Explain them Ex. console.log() console.warn(). etc...

Console in javascript is usually used for testing and debugging the code.

Following are the various methods in console function :-

  • console.log() : This method is used to log or to print the data in the console. Inside the parentheses() one can enter any type odf datatype like string,array,object,boolean,etc.

    Eg.console.log('lets upgrade');

    O/P : 'lets upgrade'

    console.log(1,2,3);

    O/P : 1,2,3

    console.log(true);

    O/P : true

    console.log(null);

    O/P : null

    console.log(undefined);

    O/P : undefined

    console.log([apple,mango,guava,strawberry,bannana]); // array inside log

    O/P : (5)[apple,mango,guava,strawberry,bananna]

    console.log({name:'sans',age:20,course:'javascript',batch number:1 }); // object inside log

    O/P :{name:'sans',age:20,course:'javascript',batch number:1 }

  • console.error() : Used to log error message to the console. Useful in testing of code. By default the error message will be highlighted with red color. console.error("this is an error notification"); by using this method the console displays this error notification.
  • console.warn() : Used to log warning message to the console. By default the warning message will be highlighted with yellow color. console.warn("this is a warning notification!"); by using this console method and calling it , it will display the warning meassage in the console.
  • console.clear() : Used to clear the console. The console will be cleared, in case of Chrome a simple overlayed text will be printed like : â€˜Console was cleared
  • console.table() : This method allows us to generate a table inside a console. The input must be an array or an object which will be shown as a table.
